# Malicious URL Detection Using Machine Learning

This project focuses on detecting malicious and phishing URLs using Machine Learning techniques to improve web security and protect users from online fraud. The system analyzes various URL characteristics such as lexical features, host-based features, site popularity features, and safe browsing information to determine whether a URL is legitimate or malicious.

The project implements and compares multiple machine learning algorithms including Random Forest, Logistic Regression, and Gradient Boosting Classifier. After evaluating the models, the Gradient Boosting Classifier achieved the best performance with an accuracy of approximately **98%** in detecting malicious URLs.

A user-friendly GUI developed using Python's Tkinter library allows users to enter a URL and instantly receive a prediction indicating whether the website is safe or malicious. This solution provides an effective approach for preventing phishing attacks and enhancing cybersecurity awareness among internet users.
